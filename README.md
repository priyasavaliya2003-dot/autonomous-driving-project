# autonomous-driving-project

## 📌 Project Overview
This project focuses on developing a basic autonomous driving system.

## 🎯 Tasks
- Implement lane detection using computer vision
- Develop object detection for obstacles
- Process sensor data (camera/LiDAR)
- Test system in simulation environments

## 🧠 Technologies Used
- Python
- Machine Learning
- Computer Vision

## 📊 Results
- Achieved reliable lane detection in test scenarios
- Improved obstacle detection accuracy
